Put the necessary sprites in the sprites folder. It is important that they have the ".png" extension and are the same size.
Then run the file "start.exe " and enter everything that the program asks for.
When the console closes, it means that the program is completed and the final file will be in the result folder.
Attention: if you run the program again without removing the previous file from the result folder, the program will replace the previous file with a new one without warning.
